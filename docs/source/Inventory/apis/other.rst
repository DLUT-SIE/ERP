^^^^^^^
    返回列表

    :get:`GET` :code:`/api/warehouses/`

    ============================ =========== ============================
    参数                            参数类型    描述
    ---------------------------- ----------- ----------------------------
    page(Optional)                  int         指定页数, 默认值: 1
    ---------------------------- ----------- ----------------------------
    limit(Optional)                 int         每页数量
    ---------------------------- ----------- ----------------------------
    name(Optional)                  str         名称
    ---------------------------- ----------- ----------------------------
    location(Optional)              str         位置
    ---------------------------- ----------- ----------------------------
    category(Optional)              int         材料类型
    ============================ =========== ============================


    ====================== =========== ============================
    返回字段                返回类型    描述
    ---------------------- ----------- ----------------------------
    id                      int
    ---------------------- ----------- ----------------------------
    name                    str         名称
    ---------------------- ----------- ----------------------------
    location                str         位置
    ---------------------- ----------- ----------------------------
    category                int         材料类型
    ---------------------- ----------- ----------------------------
    pretty_category         str         材料类型(中文)
    ====================== =========== ============================
