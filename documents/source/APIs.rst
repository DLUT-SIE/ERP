API总览
============

* 列表分页相关变量

    =================== =========== ==============================
    返回字段            返回类型    描述
    ------------------- ----------- ------------------------------
    count               int         同类对象总条目数量
    ------------------- ----------- ------------------------------
    next                str         下一页链接, 若不存在返回null
    ------------------- ----------- ------------------------------
    previous            str         上一页链接, 若不存在返回null
    ------------------- ----------- ------------------------------
    results             list        指定页面所有条目列表, 每个元素为一个字典对象
    =================== =========== ==============================
