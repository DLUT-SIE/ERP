工艺模块-API¶

工艺库¶

list¶

返回当前工艺库对应的产品信息列表

GET /api/process_libraries/

参数 参数类型 描述
page(Optional) int 指定页数, 默认值: 1
limit(Optional) int 每页数量
work_order_uid(Optional) str 指定模糊查询的工作令
返回字段 返回类型 描述
id int 工艺库id
work_order_uid int 工作令编号
name str 产品名称
writer str 编制人
proofreader str 审核人
status int {0:’未导入’,1:’未审核’,2:’审核完成’}

create¶

创建工艺库,并导入工艺物料

POST /process_libraries/upload/

参数 参数类型 描述
id int 工艺库id
file file 上传文件

partial_update¶

更新指定id的工艺库, 返回更新后的工艺库

PATCH /api/process_libraries/{id}/

参数 参数类型 描述
id int 工艺库id
返回字段 返回类型 描述
id int 工艺库id
name str 产品名称
writer str 编制人
proofreader str 审核人
status int {0:’未导入’,1:’未审核’,2:’审核完成’}
